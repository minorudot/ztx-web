# ZTX Crypt
A simple, lightweight web encrypter.
- Encrypt and decrypt files.
- Select encryption level.
- Compression.
- Use random filename or timestamp.
- View file content after decryption.

[Encrypt tool](https://minorudot.github.io/ztx-web/encrypt.html) or
[Decrypt tool](https://minorudot.github.io/ztx-web/decrypt.html)
